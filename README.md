Project description

Install
-------
```commandline
pip install client_bank_1c
```

Development
-----------
We need installed `pyenv` and `pipenv`.
```console
git clone git@github.com:odoo-ru/client-bank-1c.git

cd client-bank-1c
pipenv install --dev
```

Run tests:
```console
pipenv run fulltest
```
